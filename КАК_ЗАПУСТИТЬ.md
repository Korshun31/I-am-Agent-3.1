# Как посмотреть приложение I am Agent

## Вариант 1: В браузере (проще всего)

1. Откройте **Терминал** (Terminal) на Mac.
2. Перейдите в папку проекта:
   ```bash
   cd "/Users/igorkorshunov/I am agent Cursor"
   ```
3. Установите зависимости (один раз):
   ```bash
   npm install
   ```
4. Запустите приложение в веб-режиме:
   ```bash
   npx expo start --web
   ```
5. Откроется браузер с адресом вроде `http://localhost:8081` — там будет экран прелоудера.

---

## Вариант 2: На телефоне (Expo Go)

1. Установите на телефон приложение **Expo Go**:
   - [App Store (iPhone)](https://apps.apple.com/app/expo-go/id982107779)
   - [Google Play (Android)](https://play.google.com/store/apps/details?id=host.exp.exponent)
2. В терминале в папке проекта выполните:
   ```bash
   npm install
   npx expo start
   ```
3. В терминале появится **QR-код**. Отсканируйте его камерой iPhone или приложением Expo Go на Android.
4. Приложение откроется на телефоне.

---

## Вариант 3: В симуляторе iPhone (нужен Xcode)

1. Установите **Xcode** из App Store (если ещё нет).
2. В терминале:
   ```bash
   cd "/Users/igorkorshunov/I am agent Cursor"
   npm install
   npx expo start
   ```
3. Когда появится меню Expo, нажмите клавишу **`i`** — откроется симулятор iPhone с приложением.

---

**Итог:** для быстрого просмотра используйте **Вариант 1** (`npx expo start --web`).
