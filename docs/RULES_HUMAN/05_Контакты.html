Модуль 5: Контакты

I am Agent — Кодекс v2.0
Что делает этот модуль: управление контактами — клиенты (арендаторы) и собственники (владельцы недвижимости). Создание, редактирование, удаление, фото, документы.

5.1 Два типа контактов

✅
Клиент — арендатор, снимает жильё. Собственник — владелец недвижимости. Тип задаётся при создании и не может быть изменён.
✅
Обязательное поле: Имя. Всё остальное — по желанию: фамилия, телефон, email, Telegram, WhatsApp, паспорт, национальность, дата рождения, фото, документы.
✅
Отдельного разрешения для работы с контактами нет. Агент всегда может создавать, редактировать и удалять свои контакты.

5.2 Редактирование и удаление

✅
Админ — редактирует и удаляет любой контакт. Агент — только свои (которые сам создал).
✅TD-105
Закрыт 2026-04-30. На обеих платформах: при удалении собственника, у которого есть объекты, показывается предупреждение «У этого собственника N объектов. Если удалить контакт, объекты останутся без собственника. Удалить всё равно?».
✅TD-106
Закрыт 2026-04-30. На обеих платформах: при удалении клиента, у которого есть бронирования, показывается аналогичное предупреждение с количеством броней.
✅
При удалении контакта объекты и бронирования не удаляются — только теряют привязку к контакту.

5.3 Видимость

✅
Админ видит все контакты компании.
✅
Агент видит: (1) контакты которые сам создал, (2) собственников объектов за которые отвечает, (3) клиентов из бронирований за которые отвечает.
✅TD-099
Закрыт 2026-04-28 миграцией 20260428000001. Добавлена политика contacts: agent reads booking clients — агент видит клиента, если у него есть бронь на этого клиента.

5.4 Фото и документы

✅TD-100
Закрыт 2026-04-30. Мобильное приложение сжимает фото-аватар контакта до 1200 пикселей JPEG 85% — файл уменьшается в 10-20 раз без потери качества. Веб-аватар пока не реализован (TD-104), сжатие там появится автоматически когда добавим поле. Маленькие миниатюры (150px для списков) — отдельная задача TD-064.
✅TD-103
Закрыт 2026-04-30. На мобильном в карточке контакта появился блок «Документы»: можно прикреплять фото паспорта, договора и других документов из галереи. Каждое фото автоматически сжимается до 1200 пикселей. Можно удалять документы по крестику. Паритет с вебом.
✅TD-104
Закрыт 2026-04-30. На вебе теперь есть круглый блок аватара контакта с загрузкой, превью и удалением. Фото автоматически сжимается до 1200 пикселей качество 85% — паритет с мобильным.
