# 📋 История работы — I am Agent CRM

Папка `docs/История работы/` — оперативный журнал команды и AI-агентов.  
Отдельный контур от `docs/Устав компании/`. Устав — это правила. История — это факты.

---

## Цель

1. **Видеть историю действий "как на ладони"** — что делалось, когда, с каким результатом.
2. **Обучать новых агентов** — по журналам можно восстановить контекст без чтения всего кода.
3. **Находить лишние действия и повторяющиеся ошибки** — если одна и та же ошибка встречается 3 раза, это сигнал для системного фикса.

---

## Принципы записей

- **Кратко, но по сути:** зачем делалось → что получилось → вывод. Не "сделал X", а "сделал X потому что Y, получилось Z".
- **Без секретов:** никаких токенов, паролей, API-ключей в журналах.
- **Причинно-следственная связь:** "после X сломалось Y" — это полезно. "Внёс правку" — нет.
- **Запись без поля "результат" или "вывод" считается неполной.**

---

## Частота обновления

| Событие | Что обновить |
|---------|-------------|
| Каждый час работы | `00_Краткий_лог.md` — новая строка |
| При ошибке или инциденте | `01_Ошибки_и_фиксы.md` |
| При принятии нетривиального решения | `02_Решения_за_день.md` |
| После каждого значимого промта агенту | `03_Промты_агентам.md` |
| В конце рабочего дня | `04_Метрики_дня.md` |

---

## Обязательные файлы за день

| # | Файл | Ответственный | Статус |
|---|------|--------------|--------|
| 00 | `00_Краткий_лог.md` | Все (AI + человек) | Обязателен |
| 01 | `01_Ошибки_и_фиксы.md` | AI-агент при ошибке | По факту |
| 02 | `02_Решения_за_день.md` | Architect / Developer | По факту |
| 03 | `03_Промты_агентам.md` | Человек (владелец) | По факту |
| 04 | `04_Метрики_дня.md` | Человек | В конце дня |

---

## Как начать новый день

1. Скопировать папку `Шаблон_дня/` → переименовать в `YYYY-MM-DD/`
2. Заполнить заголовки (дата, задачи на день)
3. Обновлять `00_Краткий_лог.md` в течение дня

---

## Правило качества

> Запись, в которой нет поля **"Результат"** и **"Следующий шаг"** (или **"Урок"** в ошибках) — считается **неполной** и не несёт ценности для команды.
